# 今すぐ起動する（ZIP 展開済み・セットアップ済み）

このフォルダは **ZIP を展開した状態** で、`C:\Users\green\yt-shorts-app` にあります。

## 起動までの手順

### 1. client_secrets.json を置く（初回だけ）

- **secrets** フォルダを開く
- Google Cloud で取得した **client_secrets.json** を、そのまま **secrets** フォルダの中に保存する  
  （ファイル名は `client_secrets.json` のまま）
- まだ取得していない場合は「はじめて使う人向け.md」の「4. 設定ファイルを用意する」を参照

### 2. アプリを起動する

**方法A（おすすめ）**

- **start_app.bat** をダブルクリック

**方法B（ターミナルから）**

```powershell
cd C:\Users\green\yt-shorts-app
python run_app.py
```

### 3. ブラウザを開く

- **http://localhost:8501** を開く  
  （起動後、自動で開く場合もあります）

---

## 初回だけ

- 画面上の **「はじめての方」** で config とアカウントの状態を確認
- **「アカウント」** で「アカウントを追加」→ ブラウザで Google にログインして権限を許可
- **「キュー」** で動画を1本追加してから **「アップロード実行」** でドライラン → 本実行

---

## トラブル時

- **「encodings が見つからない」**  
  → フォルダを **C:\yt-shorts-app** など、パスに日本語が含まれない場所にコピーしてから同じ手順で起動
- **「client_secrets が見つかりません」**  
  → **secrets** フォルダ内に **client_secrets.json** があるか確認
