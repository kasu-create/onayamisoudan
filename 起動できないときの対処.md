# 起動できないときの対処

## 1. 「encodings が見つからない」エラーについて

**このエラーは Python 本体の不具合です。** プロジェクトのフォルダの場所ではなく、**PC にインストールされている Python が正しく動いていません。**

### 対処: Python の修復または再インストール

1. **コントロールパネル** → **プログラムのアンインストール** を開く  
2. **Python 3.12** を探す → **変更** を選び、**「Repair（修復）」** を実行  
3. それでもダメな場合: 一度 **アンインストール** し、[python.org](https://www.python.org/downloads/) から最新版を入れ直す  
   - インストール時に **「Add Python to PATH」に必ずチェック** を入れる  
4. **Microsoft Store 版の Python** を試す方法もあります（Store で「Python 3.12」を検索してインストール）

修復・再インストール後、**新しいターミナル**を開き直してから再度実行してください。

---

## 2. パスに日本語が含まれる場合

このツールは **パスに日本語（マイドライブなど）が含まれるフォルダ** から実行すると、encodings エラーになることがあります。

### 対処

1. このフォルダを **C:\yt-shorts-app** や **D:\ShortsApp** など、**英数字のみのパス** にコピーする  
2. ターミナルで **cd （そのフォルダのパス）** してから **python run_app.py** を実行する  
3. または **start_app.bat** をダブルクリック（バッチがこのフォルダをカレントにしてから実行します）

---

## 3. まとめ

| 症状 | 対処 |
|------|------|
| encodings が見つからない | Python の修復または再インストール |
| パスに日本語がある | フォルダを C:\yt-shorts-app などにコピーしてから実行 |
| それでも起動しない | ターミナルで **cd （フォルダのパス）** してから **python run_app.py** を実行 |

エラーが出た場合は、ターミナルに表示された**エラー全文**をコピーしてサポートに伝えると原因を特定しやすくなります。
