========================================
配布用フォルダ — 使い方
========================================

■ このフォルダに含まれているもの

  - アプリ本体（app.py, run_app.py, add_account.py, run_upload.py, upload.py, auth.py, quota.py など）
  - 設定サンプル（config.example.yaml, upload_queue.example.csv）
  - 説明書（はじめて使う人向け.md, README.md, 運用のしかた, 起動方法.txt など）
  - .cursor/rules/ … Cursor がプロジェクトを理解するためのルール
  - Cursor向け説明書.md … 受け取った人向けの「Cursor で開いてどう使うか」の説明
  - secrets/（client_secrets は空。受け取り手が自分で置く）
  - accounts/（空。受け取り手がアカウント追加で使う）

■ 配布する人：渡す前にやること

  1. accounts/ 内に token が入ったフォルダがある場合は削除する（個人の権限のため渡さない）。
  2. secrets/ に client_secrets.json がある場合は削除する（受け取り手が自分で置く）。
  3. この「配布用」フォルダ一式を ZIP にして渡す。

■ 受け取った人：最初にやること

  1. ZIP を解凍して、できたフォルダを Cursor で開く。
  2. 「Cursor向け説明書.md」を開いて読む。
  3. 説明書のとおり、AI に「起動方法を教えて」「セットアップ手順を教えて」などと聞く。
  4. フォルダのパスに日本語が含まれる場合は、C:\yt-shorts-app などにコピーしてから実行する。

■ ポイント

  - このフォルダを Cursor で開くだけで、AI がプロジェクトの前提・起動方法・構成を理解します。
  - 設定やアカウント追加・キュー追加は、アプリの Web UI から行えます（Python を入れないで exe で動かす場合は、exe の案内に従ってください）。
