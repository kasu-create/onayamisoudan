# はじめて使う人向け - セットアップ手順

このツールを初めて使うときは、次の順番で進めてください。

---

## 1. Python を入れる

- [python.org のダウンロード](https://www.python.org/downloads/) からインストーラーを取得
- インストール時に **「Add Python to PATH」にチェック** を入れる
- インストール後、新しいターミナル（PowerShell や コマンドプロンプト）を開き、次で確認:
  ```bash
  python --version
  ```
  （例: `Python 3.12.x` と出ればOK）

**注意**: パスに日本語が含まれるフォルダ（例: マイドライブ）で実行するとエラーになることがあります。このツールは **C:\yt-shorts-app** や **D:\ShortsApp** のように、**英数字のみのパス** に置いて使うことをおすすめします。

---

## 2. このフォルダを置く

- このフォルダを、使いたい場所にコピーする  
  例: `C:\yt-shorts-app` や `D:\ShortsApp`
- **フォルダのパスに日本語（マイドライブなど）が含まれないようにする**

---

## 3. パッケージをインストールする

ターミナルで **このフォルダに移動** してから実行:

```bash
cd （このフォルダのパス。例: C:\yt-shorts-app）
python -m pip install -r requirements.txt
```

（`pip` が認識されない場合は、必ず `python -m pip` を使ってください。）

---

## 4. 設定ファイルを用意する

1. **config.yaml**  
   - `config.example.yaml` をコピーして `config.yaml` という名前で保存する  
   - 中身はそのままでOK（必要なら後から編集）

2. **client_secrets.json**  
   - Google Cloud で「YouTube Data API v3」を有効にしたプロジェクトを作成
   - 「認証情報」→「認証情報を作成」→「OAuth クライアント ID」→ アプリの種類は **デスクトップ**
   - 作成後「JSON をダウンロード」し、ファイル名を **client_secrets.json** に変更
   - このフォルダ内の **secrets** フォルダに保存する

OAuth の「同意画面」で「外部」を選んだ場合は、テストユーザーに自分の Google アカウントを追加してください。

---

## 5. アカウントを1つ登録する

ターミナルで（**このフォルダがカレント**の状態で）:

```bash
python add_account.py account_01
```

- ブラウザが開くので、**YouTube に使う Google アカウント**でログイン
- 権限を「許可」すると、`accounts\account_01` にトークンが保存されます  
- 2つ目以降は `account_02`、`account_03` … のように名前を変えて同じコマンドを実行

---

## 6. アプリを起動する

- **start_app.bat** をダブルクリック  
  または ターミナルで:
  ```bash
  cd （このフォルダのパス）
  python run_app.py
  ```
- ブラウザで **http://localhost:8501** を開く

---

## 7. 動画をキューに追加して実行する

1. アプリの **「キュー」** で、表に1行追加する  
   - **account_id**: 登録した名前（例: `account_01`）  
   - **video_path**: 動画ファイルのパス（例: `C:\Videos\short1.mp4`）  
   - **title**, **description** など必要項目を入力  
   - **status**: `待機` のまま
2. **「アップロード実行」** で、まず **ドライラン** にチェックを入れて実行し、問題なければドライランを外して本実行

---

## 困ったとき

- **「encodings が見つからない」**  
  → Python の不具合です。Python の修復または再インストール（「Add to PATH」を忘れずに）を試してください。README.txt にも手順があります。

- **起動しない・パスが分からない**  
  → フォルダを **C:\yt-shorts-app** など、パスに日本語が含まれない場所にコピーしてから、同じ手順で起動してください。

- **ほかの人が別のパソコンからアカウントを追加したい**  
  → **運用のしかた（動画準備・他PCからアカウント追加）.md** を参照してください。
